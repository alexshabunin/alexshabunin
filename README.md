### 🧭 About

~4 years in IT — grew from **Manual QA to 100% AQA**. I build test frameworks
from scratch and mentor manual testers into automation. I treat tests as
**architecture, not checklists**: risk-based strategy, traceable coverage, and a
CI signal that's honest about green.
